Implemented in Processing using <a href="http://glgraphics.sourceforge.net/">GLGraphics<a/> and <a href="http://code.google.com/p/proscene/">Proscene</a> libraries. Code available <a href="https://github.com/genekogan/FlockingBoids">here</a>.
