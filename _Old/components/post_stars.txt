<p>
	Using Processing to generate fancy stars. We all know how to make the familiar five pointed, straight-edged star: Make five points around a circle, and connect the dots by skipping a single point each time. Letting the number of points and number of skips be variables, and turning the straight lines into bezier curves, a wide variety of stars can be produced. <a href="http://www.openprocessing.org/sketch/65386">Processing code available here</a>.
