<p>
	A project combining Photoshop's abundance of preset filters and blending strategies with its handy and little-known scripting capabilities. To generate these, I first collected a set of textures via Google image search to use as source images. I then wrote a script which automated the following procedure:
</p>
<p>
	<ul>
		<li>Select five textures at random</li>
		<li>Apply one of Photoshop's filters to each texture, selecting the filter and its parameters randomly.</li>
		<li>Apply an alpha mask along a contour created by clicking the magic wand tool somewhere on the texture, giving the texture some transparency.</li>
		<li>Blend the five warped textures together into a single image, using five randomly chosen blending modes.</li>
	</ul>
</p>
