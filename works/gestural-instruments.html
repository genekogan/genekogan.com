Gestural Instruments is an ongoing project attempting to develop novel musical instruments with the Leap Motion hand tracker. The Leap Motion is a small device providing extremely precise hand tracking, opening up potential new ways of interacting with media, including as a musical instrument controller.

I wrote several instruments using SuperCollider and set the Leap Motion to control the synth parameters. The code for these instruments can be found on GitHub. Some highlights of those experiments:

Maceda Box was an experimental installation for interacting with the field recordings of Filipino ethnomusicologist, Jose Maceda, created as part of my residency with The UnifiedField in Davao City, Phillipines. A collection of Maceda's recordings, acoustic fragments spanning the whole of the Philippines and drawing from its varied musical traditions, are spatially organized according to acoustic similarity, allowing a user to draw upon the works in an intuitive and playful way with the use of the Leap Motion.
