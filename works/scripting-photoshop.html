A project in creating generative images using Photoshop's little-known scripting capabilities. A series of textures were collected via Google image search to use as source images. The following procedure is then automated using a script, and applied iteratively.

  • 1) Select five textures at random.
  • 2) Apply one of Photoshop's filters to each texture, selecting the filter and its parameters randomly.
  • 3) Apply an alpha mask along a contour created by clicking the magic wand tool somewhere on the texture, giving the texture some transparency.
  • 4) Blend the five warped textures together into a single image, using five randomly chosen blending modes.
