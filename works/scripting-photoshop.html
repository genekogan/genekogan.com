Scripting Photoshop

Generating textures with Photoshop's scripting functions. A series of textures were collected as source images, and the following procedure was applied iteratively.

  • 1) Select five textures
  • 2) Apply a random filter to each
  • 3) Alpha mask each along a contour from a random magic wand selection
  • 4) Blend the five results into a single image
