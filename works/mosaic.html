Mosaic is a program that facilitates the creation of the classic photo mosaic, a large composition of many small photographs stitched together in such a way as to resemble a single target photograph.

The software was written in Matlab and is designed to take as input a large collection of images, and a single "target image", i.e. the photo which the mosaic should model. It works by dividing the target into a grid, and then fitting individual photographs into each "pixel" within the grid, so as to maximize the overall color similarity between source and target, using an implementation of Earth Mover's distance.

The full code for this project is available on GitHub, though requires Matlab to use it. Someday when I get around to it, I will make a version in a free language like Python.
