Kinect Projector Toolkit

Kinect Projector Toolkit is a library for Processing and OpenFrameworks which calibrates a projector to a Kinect depth camera, aligning a projection to the physical space it's lighting. Highlights from several installations:

Blair Neal using KinectProjectorToolkit for live visuals with Phantogram.

The toolkit is free and open source. It contains an app for calibration, and a performance application for applying effects onto calibrated stages.

The Processing version also comes with a tutorial for calibrating.
