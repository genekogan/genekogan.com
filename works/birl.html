Birl

The Birl is an electronic flute built by Jeff Snyder at Snyderphonics. It encases a custom circuit with a set of finger pressure sensors, along with a breath pressure and embouchure sensor on the mouthpiece.

We are developing an application which interfaces with the Birl, allowing a way to easily train it using machine learning, creating customized relationships between the sensor data and the synthesis parameters of a physical model. The software sends synthesis parameter data over OSC, and in the future will control synthesis directly on the microcontroller.
